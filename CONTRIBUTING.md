# Contributing

Thank you for helping build Context Continuum.

1. Read `AGENTS.md` and the canonical PSPR before changing code.
2. Keep changes scoped to one CAC prompt or one clearly named issue.
3. Preserve the exact `gpt-5.6-sol`-only contract; model fallbacks are out of scope.
4. Add tests and reproducible evidence for changed behavior.
5. Run the baseline Rust checks from `docs/VERIFICATION.md`.
6. Never commit credentials, private transcripts, generated databases, or unsanitized evidence.

By contributing, you agree that your contribution is licensed under Apache-2.0.
